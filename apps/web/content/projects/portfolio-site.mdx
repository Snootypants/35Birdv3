# Portfolio Website

This portfolio website showcases my projects, blog posts, and creative works.

## Features

- **Blog System**: MDX-powered blog with syntax highlighting
- **Project Showcase**: Dynamic project pages with metadata
- **Interactive Games**: Browser-based games built with React
- **Admin Dashboard**: Content management interface
- **Gallery**: Visual works and screenshots

## Tech Stack

- Next.js 14 with App Router
- TypeScript for type safety
- Tailwind CSS for styling
- Supabase for authentication and database
- MDX for content management
- Turborepo for monorepo management

## Architecture

The site uses a monorepo structure with separate packages for:
- Web application
- UI components library
- Game packages
